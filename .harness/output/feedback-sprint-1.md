## 改进建议 ⚠️

### P2 - 功能完善

#### 1. Supabase 后端配置
**问题**: 认证功能使用 Mock 数据，未连接实际后端  
**建议**: 
- 创建 Supabase 项目
- 配置 `VITE_SUPABASE_URL` 和 `VITE_SUPABASE_ANON_KEY`
- 创建数据库表（users, photos, view_stats）

#### 2. 图片上传功能
**问题**: 上传页面仅为前端展示  
**建议**:
- 集成 Supabase Storage
- 实现文件选择和预览
- 添加上传进度和错误处理

### P2 - 代码优化

#### 3. 单元测试
**问题**: 缺少测试覆盖  
**建议**:
- 安装 Vitest + React Testing Library
- 为 Button、Input 等基础组件编写测试
- 添加认证流程的集成测试

### P2 - 设计优化

#### 4. 动画效果
**问题**: 部分过渡动画可以更流畅  
**建议**:
- 使用 `transform` 和 `opacity` 实现 GPU 加速
- 优化 transition 时序函数
- 添加页面加载骨架屏
