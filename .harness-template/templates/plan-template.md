```
访问首页 → 感受产品价值 → 注册 → 首次操作 → 体验核心功能 → 留存
```

```
打开应用 → 浏览内容 → 互动（点赞/评论/收藏） → 创作内容 → 查看数据 → 离开
```

**旅程 3：内容创作者**
```

```

| 后端 | Supabase | 2.x | Auth+DB+Storage 一站式 |
